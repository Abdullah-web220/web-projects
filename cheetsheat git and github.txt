git commit – creates a new commit

-a – automatically stages all modified and deleted files that are already tracked

❌ does not include new (untracked) files

-m "adding about page" – sets the commit message

Important gotcha

If about page was a new file, it will not be included in this commit.

To include new files, you’d need:

git add .
git commit -m "adding about page"
